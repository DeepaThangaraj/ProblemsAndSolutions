how to remove port 8080 running already.couldnot start the process while running a spring boot application in STS,Eclipse

go to cmd
type
C:\WINDOWS\system32>netstat -aon | find /i "listening"
hit enter
apply the port filter
type
C:\WINDOWS\system32>netstat -aon |find /i "listening" |find "8080"
hit enter
note the PID
type
C:\WINDOWS\system32>taskkill /F /PID 4956
